# Autonomous AI DeFi Agent

## Overview
This project is a dashboard and autonomous agent system for a DeFi "Hedge Fund" that optimizes yields via the Compound Effect.

## Structure
- `/packages/contracts`: Smart contracts (Foundry)
- `/packages/agents`: AI Agents (TypeScript/Node.js)

## Prerequisites
- **Node.js** (v18+)
- **Foundry** (Forge): Required for smart contracts.
    - Install: `curl -L https://foundry.paradigm.xyz | bash`

## Getting Started
1. Install dependencies: `npm install`
2. Run agents: `npm run start:agent`
